SharePoint Setup Required
One-time configuration to enable shared team storage
📋 This tool stores architecture artefacts in a SharePoint List so your entire team can view and edit them from any browser — no installation required.
1
Create the SharePoint List
Go to your SharePoint site → Site Contents → New → List. Name it exactly: ArchitectureRepository

Or ask your SharePoint admin to run this PowerShell script:
Connect-PnPOnline -Url "https://YOURTENANT.sharepoint.com/sites/YOURSITE" -Interactive New-PnPList -Title "ArchitectureRepository" -Template GenericList Add-PnPField -List "ArchitectureRepository" -DisplayName "ArtefactId" -InternalName "ArtefactId" -Type Text Add-PnPField -List "ArchitectureRepository" -DisplayName "ArchDomain" -InternalName "ArchDomain" -Type Text Add-PnPField -List "ArchitectureRepository" -DisplayName "Subdomain" -InternalName "Subdomain" -Type Text Add-PnPField -List "ArchitectureRepository" -DisplayName "ArtefactType" -InternalName "ArtefactType" -Type Text Add-PnPField -List "ArchitectureRepository" -DisplayName "ArtefactStatus" -InternalName "ArtefactStatus" -Type Text Add-PnPField -List "ArchitectureRepository" -DisplayName "Version" -InternalName "Version" -Type Text Add-PnPField -List "ArchitectureRepository" -DisplayName "Owner" -InternalName "Owner" -Type Text Add-PnPField -List "ArchitectureRepository" -DisplayName "ArchDescription" -InternalName "ArchDescription" -Type Note Add-PnPField -List "ArchitectureRepository" -DisplayName "Rationale" -InternalName "Rationale" -Type Note Add-PnPField -List "ArchitectureRepository" -DisplayName "BusinessGroups" -InternalName "BusinessGroups" -Type Note Add-PnPField -List "ArchitectureRepository" -DisplayName "Projects" -InternalName "Projects" -Type Note Add-PnPField -List "ArchitectureRepository" -DisplayName "Tags" -InternalName "Tags" -Type Note Add-PnPField -List "ArchitectureRepository" -DisplayName "ArtefactCreated" -InternalName "ArtefactCreated" -Type Text Add-PnPField -List "ArchitectureRepository" -DisplayName "ArtefactUpdated" -InternalName "ArtefactUpdated" -Type Text
2
Upload both files to SharePoint Site Assets
Go to Site Contents → Site Assets. Upload both index.html and dashboard.js into the same folder. Both files must reside in the same SharePoint site as the list.
3
Open from SharePoint and share the link
Click index.html in Site Assets to open it in the browser. Copy the URL and share it via Teams or add it as a Teams tab. Entra ID Conditional Access (GRP-TOGAF-Users) enforces authentication automatically.
